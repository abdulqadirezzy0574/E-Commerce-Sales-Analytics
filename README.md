# E-Commerce-Sales-Analytics
A data analytics project that explores e-commerce sales and customer behavior using SQL and Power BI, providing interactive dashboards and actionable business insights.

## Project Objective
The objective of this project is to analyze e-commerce sales data using SQL and Power BI to understand customer purchasing behavior, product performance, and overall sales trends. The project focuses on identifying top-selling product categories, customer preferences, payment methods, seasonal sales patterns, and revenue trends through interactive dashboards and data analysis.

By converting raw sales data into meaningful insights, this project demonstrates the use of SQL for data analysis and Power BI for data visualization. The findings can help businesses improve customer experience, optimize inventory, increase sales, and make better business decisions based on data.
